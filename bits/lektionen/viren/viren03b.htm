Mögliche Schäden durch Virenbefall (3)

Heutzutage wird häufig über "Bot-Netze" und "gekaperte" Rechner oder "Zombies" gesprochen. Hier ist eine kurze Erklärung, wie das funktioniert:

Ein erster Angreifer streut breit einen Virus und infiziert mehrere Computer

Die infizierten ("gekaperten") Computer ("Zombies") melden sich (unbemerkt vom Benutzer) an einem Server an und bilden zusammen ein Netzwerk ("Bot-Netz")

Ein zweiter Angreifer (z.B. Spammer) mietet beim ersten einen Zugang zur Bot-Netz-Steuerung

Der zweite Angreifer erteilt dem Bot-Netz eine Aufgabe (z.B. "Versendet Werbung für XXX!")

Der gekaperte Computer führt zusammen mit den anderen Zombies verdeckt die Aufgabe aus und wartet auf weitere Befehle

  • Diebstahl von Systemfunktionen
  • Verbrauch von Systemressourcen
  • kriminelle Ziele
